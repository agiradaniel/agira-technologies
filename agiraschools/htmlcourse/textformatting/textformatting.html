  • What is text formatting?
  • Various text formats

Text formatting can be defined as giving text special attributes that make it somehow different to normal text

Text bolding makes the text bold, you can use the b tag or the strong tag, but the strong tag is recommended

Text italicizing makes the text appear in italics formatt, you can use the i tag or the em tag, but the em tag is recommended

Text underline underlines the text, you can use the u tag or the ins tag, but the ins tag is recommended

Subscript creates a small character that appears below the line, the sub tag is used

Superscript creates a small character that appears above the line, the sup tag is used

Strike through cancels a text by crossing a horizontal line,the del tag is used

Small creates a text that is smaller than normal, the small tag is used

Horizontal rule creates a horizontal line to separate some sections of a webpage, the hr tag is used
