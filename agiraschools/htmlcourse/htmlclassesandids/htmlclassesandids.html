  • What is the difference between classes and IDs

HTML classes specify or identify one or more elements in a html document, they can be used by CSS and Javascript to perform certain tasks for elements with the class name, we will interact more with classes in the CSS section

HTML IDs work in the same way as classes but they uniquely identify an element ie they cannot be used for several elements

The difference between classes and IDs is that classes can be applied to many elements but IDs only to one element
